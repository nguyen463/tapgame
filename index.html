Cara Menggunakan

  1. Pastikan Anda telah menginstal wallet Solana (seperti Phantom) di browser Anda
  2. Klik tombol "Connect Wallet" untuk menghubungkan wallet
  3. Setelah terhubung, Anda dapat memainkan game Tic-Tac-Toe
  4. Klik pada kotak untuk menempatkan simbol X, kemudian komputer akan merespons dengan O
  5. Gunakan tombol "Reset Game" untuk memulai permainan baru
